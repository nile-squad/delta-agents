# DataSources

> **Status:** Being repurposed for delta-agents. Previous @nilejs/future content is in git history.

This document should cover the DataSource authoring type from the spec: name, description, ownership (internal/external), contentType, authentication, and CRUD actions (retrieve, create, update, delete). DataSources are part of the authoring API that developers configure, distinct from the runtime API Delta owns.

See [delta-agents.spec.md](../delta-agents.spec.md) for the canonical specification.
